Section

Section 1 "¿Qué es section?"

El elemento de HTML section representa una sección genérica "Esto es un ejemplo de ello"
de un documento. Sirve para determinar qué contenido corresponde a qué parte
de un esquema. Piensa en el esquema como en el índice de contenido de un libro;
un tema común y subsecciones relacionadas. Es, por lo tanto, una etiqueta semántica.

Article

Para que usamos Article?

El elemento article puede ser utilizado en conjunto con otros elementos como header , footer
y time para agregar información acerca de su contenido. Esta información puede ser utilizada,
por ejemplo, por navegadores o lectores de noticias.

Header

Uso del Header

La etiqueta "header" en HTML se utiliza para contener información de encabezado de un documento o sección.
El contenido dentro de la etiqueta "header" suele incluir elementos como el título de la página, el logotipo,
la navegación y otros elementos de encabezado relacionados.

Nav

El elemento HTML "nav" representa una sección de una página cuyo propósito es proporcionar enlaces de navegación,
ya sea dentro del documento actual o a otros documentos.

Figure

El elemento HTML "figure" representa contenido independiente, a menudo con un título.

Figcaption

El elemento HTML "figcaption" representa un subtítulo o leyenda asociado al contenido del elemento
padre "figure" , pudiendo ser colocado como primer o último hijo.

Details

El elemento HTML Details "details" es usado como un widget de revelación a través del cual el usuario puede obtener información adicional

Summary

La etiqueta "summary" define un texto visible para el elemento details. En el texto se puede hacer clic para ver/ocultar los detalles.

Time

El elemento HTML "time" representa un periodo específico en el tiempo. Puede incluir el atributo datetime
para convertir las fechas en un formato interno legible por un ordenador, permitiendo mejores resultados en
los motores de búsqueda o características personalizadas como recordatorios.

Open from 10:00 to 21:00 every weekday.

I have a date on Valentines day.

Aside

El elemento HTML "aside" representa una sección de una página que consiste en contenido que está
indirectamente relacionado con el contenido principal del documento "En este ejemplo se encuentra en el footer".
